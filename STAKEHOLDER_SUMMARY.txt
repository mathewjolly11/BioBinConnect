================================================================================
                    STAKEHOLDER 4: COMPOST MANAGER
================================================================================

ROLE: Waste Processor & Compost Producer

WHAT THEY DO:
1. Register with license number
2. View available waste from collectors
3. Create compost batches from source waste
4. Monitor composting process
5. Grade compost quality (Premium/A/B/C)
6. Set pricing for compost batches
7. Update batch status (Processing → Ready → Sold)
8. Track earnings based on compost produced

KEY RESPONSIBILITIES:
✓ Convert waste to high-quality compost
✓ Quality control and grading
✓ Stock management
✓ Pricing strategy
✓ Batch tracking

BATCH CREATION PROCESS:
1. Select available waste (e.g., 160kg)
2. Enter batch name (e.g., "Batch AA")
3. System calculates compost output: 160kg ÷ 4 = 40kg
4. Select grade (Premium/A/B/C)
5. Set price (e.g., ₹200/kg)
6. Create batch with status "Processing"
7. Update to "Ready" when composting complete
8. Auto-updates to "Sold" when farmer orders

CONVERSION RATIO:
- Default: 4kg waste → 1kg compost
- Configurable by admin

EARNINGS:
- Rate: ₹1,000 per day worked
- Example: Work 5 days → Earn ₹5,000
- Payment processed by admin

GRADING & PRICING:
- Premium: ₹250-300/kg (highest quality)
- Grade A: ₹200-250/kg
- Grade B: ₹150-200/kg
- Grade C: ₹100-150/kg

ACCESS: Compost Manager portal via /compost-manager/ URL

================================================================================
                    CROSS-STAKEHOLDER INTERACTIONS
================================================================================

FLOW 1: Household → Collector → Admin
--------------------------------------
Household requests pickup → Collector collects waste → Logs in system →
Inventory updated → Household pays monthly fee → Admin tracks payment

FLOW 2: Collector → Compost Manager → Farmer
---------------------------------------------
Collector has leftover waste → Manager creates compost batch →
Marks ready for sale → Farmer orders compost → Admin processes →
Manager delivers → Farmer pays → Admin pays manager salary

FLOW 3: Collector → Farmer (Direct Waste Sale)
-----------------------------------------------
Collector collects waste → Adds to inventory → Farmer orders waste →
Admin assigns collector → Collector delivers → Farmer pays →
Admin pays collector salary
