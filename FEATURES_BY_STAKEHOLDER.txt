STEP 8: Deploy

EXAMPLE: Adding Route Optimization for Collector

1. Edit: CollectorApp/views.py
   ```python
   def optimized_pickups(request):
       pickups = tbl_PickupRequest.objects.filter(
           collector=request.user.collector,
           status='Pending'
       ).order_by('household__house_no')  # Sort by house number
       return render(request, 'collector/optimized_pickups.html', 
                    {'pickups': pickups})
   ```

2. Edit: CollectorApp/urls.py
   ```python
   path('optimized-pickups/', views.optimized_pickups, 
        name='optimized_pickups'),
   ```

3. Create: templates/collector/optimized_pickups.html
   (Display pickups in optimized order with map)
